To add new subtree:
    git remote add [REMOTE_NAME] git@github.com:[USER_NAME]/[REPO_NAME].git
    git subtree add --prefix=[PATH_TO_LOCAL_SUBTREE] --squash [REMOTE_NAME] master

To update subtree:
    git subtree pull --prefix=[PATH_TO_LOCAL_SUBTREE] --squash [REMOTE_NAME] master
